# The Bella Wildflower

Floral Design & Experiences — artisanal wedding and event floristry.

## Tech Stack
- Next.js 14.2.33
- TypeScript
- Tailwind CSS
- Backend: Base44 (shared with ReEmpoderate ecosystem)
